# Калькулятор выбора сечения кабеля по нагреву

Однофайловый HTML-калькулятор (`index.html`). ПУЭ 7 изд., Гл.1.3, ГОСТ Р 50345-2010, ГОСТ 31996-2012.

## Что считает

Подбор сечения медного/алюминиевого кабеля по допустимому нагреву + выбор автомата защиты.

## Алгоритм

1. **Расчётный ток**: Iр = Iн × Kи
2. **Подбор сечения**: ищем минимальное, где I_д.д × Kt × Kb × Kg × Kж ≥ Iр
3. **Выбор автомата**: ближайший стандартный ≥ Iр (ГОСТ Р 50345)
4. **Проверки**:
   - I_авт ≤ I_д.д.р
   - 1.13 × I_авт ≤ перегрузочный ток (ГОСТ 31996,空气 1.16, земля 1.13)
   - Если не выполняется → берём большее сечение

## Входные параметры

| Поле | Описание |
|------|----------|
| Напряжение, В | 220 / 380 / другое |
| Номинальный ток Iн, А | Ток нагрузки (или через мощность) |
| Мощность P, кВт | Если ток неизвестен. I = P×1000 / (√3×U×cosφ) |
| cos φ | Коэф. мощности (по умолчанию 0.85) |
| Kи | Коэф. использования (0.1–1.0) |
| Материал жилы | Медь / Алюминий |
| Изоляция жилы | ПВХ (70°C) / Резина (65°C) / XLPE (90°C) |
| Число жил | 1–5. Для 4–5: Kж = 0.93 |
| Среда прокладки | Воздух / Земля |
| Способ прокладки | Открыто / Лоток / Труба |
| Кол-во кабелей | Для пучков/лотков |
| Температура среды, °C | Воздуха или земли |
| Расст. между кабелями | 100/200/300 мм (только для земли) |

## Коэффициенты

| Обозначение | Источник | Применение |
|-------------|----------|------------|
| Kt | Табл. 1.3.3 ПУЭ | Коррекция на температуру среды |
| Kb | Табл. 1.3.12 ПУЭ | Снижение для прокладки в лотках/коробах |
| Kg | Табл. 1.3.26 ПУЭ | Совместная прокладка в земле |
| Kж = 0.93 | ГОСТ 31996-2012 | 4–5 жильные кабели |

Kt зависит от среды: воздух (эталон 25°C) vs земля (эталон 15°C). Таблица содержит строки для обеих.

## Табличные данные (захардкожены в JS)

- **Табл. 1.3.6** — Медь, пластмассовая/резиновая изоляция. Столбцы: 1-жил.воздух, 2-жил.воздух/земля, 3-жил.воздух/земля
- **Табл. 1.3.7** — Алюминий, аналогично
- **Табл. 1.3.3** — Kt по температуре (2 группы: воздух/земля × 7 температур жилы)
- **Табл. 1.3.12** — Kb для лотков (однослойно/многослойно)
- **Табл. 1.3.26** — Kg для земли (расстояния 100/200/300 мм)

Данные извлечены из PDF `ПУЭ 6-е и 7-е изд..pdf` через PyPDF2.

## Проверенные примеры

### Пример 1 (из docx)
Вход: In=28A, Ki=0.8, Cu, 3-жил., воздух, открыто, 1 кабель, 25°C, ПВХ
→ Ir=22.4A, 2.5мм², автомат 25A ✓

### Пример 2 (из docx, без Ki)
Вход: In=28A, без Ki, Cu, 3-жил., воздух, открыто, 1 кабель, 25°C
→ Ir=28A, 4мм², автомат 32A ✓

## Файлы в папке

- `index.html` — калькулятор (единственный файл, всё включено)
- `Выбор сечения по нагреву.docx` — методичка с примером расчёта
- `ПУЭ 6-е и 7-е изд..pdf` — полный текст ПУЭ (источник таблиц)
- 50+ PDF — библиотека нормативов (ГОСТ, СП, справочники)

## Известные ограничения

- Таблицы 1.3.6/1.3.7 до 240 мм² (большие сечения не покрыты)
- Коэффициенты для кабелей в блоках не реализованы
- Проверка падения напряжения не реализована
- Нет расчёта токов КЗ
